Every Relay guitar uses the **same printed body platform** — the same architecture, neck format, and three-pickup layout. What defines a **model** is not a different body file; it is the **pickup set and electronics identity** built around that shared foundation. When you choose a model, you are choosing **tonal center** and **control story**: on some models the middle pickup is a **curated auxiliary layer** (3-way switch family); on others it is a **core voice** that deserves its own positions (5-way blade family). See each model page for the intent.

For **effort, skills, and stage-by-stage difficulty** shared by every model, read [What building takes](/relay/printing/build-expectations) next (or before you commit filament).

## Meet the lineup

Each model has a short landing page focused on **why you might want to build it** — tone, vibe, and musical context, not a parts list.

- [Relay Lipstick](/relay/lipstick) — **Available.** Expressive signature model: dual humbuckers + middle lipstick as a curated alternate voice (3-way family). The only model with a full instrument BOM, compatibility notes, and a released print bundle today.
- [Relay Velvet](/relay/velvet) — Planned. Warm, fat club authority; middle as a core voice (5-way family).
- [Relay Arc](/relay/arc) — Planned. Airy, chime-forward, spatial clarity — open and alive, not club-warm (5-way family).
- [Relay Torch](/relay/torch) — Planned. Rude vocal mids, contemporary hooky energy; P90-type middle as a core position (5-way family).
- [Relay Current](/relay/current) — Planned. Punch, cut, and immediacy — the line’s sharpest non-metal tool (3-way family, middle as fast alternate layer).
- [Relay Hammer](/relay/hammer) — Planned. Dedicated heavy model; rails, tight gain, metal focus (5-way family).

Skim the pages that speak to you. There is nothing to buy — the point is to picture the instrument you want on the stand before you commit filament and weekends.

## What exists in the docs today

**Printing and body assembly** are documented at the **platform** level: same materials, same bonding workflow, same structural expectations for every model.

**Model-specific** documentation (full parts lists, compatibility, wiring tuned to that pickup set) tracks each release. Today, **Relay Lipstick** is the only model with that depth. Planned models link forward to shared platform guides for anything that already applies.

If you are building Lipstick, continue from here to [Printing overview](/relay/printing/overview) for downloads and material notes, and use the Lipstick [Supply List](/relay/lipstick/bom) and [What Will Fit](/relay/lipstick/compatibility) before you order hardware.

If you are following a planned model, treat the platform printing and build sections as your reference for the body; watch that model’s page for updates as electronics and BOMs ship.

## Community input

The lineup on the [platform overview](/relay) includes a voting tool for **planned** models — optional feedback that helps prioritize what gets built next. It does not change what is available today.
