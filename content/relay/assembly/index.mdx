Assembly covers everything between a printed-and-finished body and a playable guitar — hardware install, wiring, intonation, and setup. It's the final stage of the Relay build process.

## What this guide will cover

- Hardware install (bridge, tuners, neck plate, jack, ground)
- Wiring per voicing — references the wiring diagram on each voicing page
- Pickup install and height adjustment
- Stringing, intonation, and action setup
- Final pass: nut work, fret leveling if needed, setup notes for the body's resonance

## Status: planned

I haven't started writing this content yet. The body stage and several voicing builds need to happen first so the assembly notes are based on what actually works. If you want to weigh in on what should be in this guide, the conversation is in Discord.
