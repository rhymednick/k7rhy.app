There is no single correct assembly sequence, but some approaches work better for beginners and others give more flexibility for troubleshooting. Here are three paths.

## Electronics-first (recommended for first builds)

Install and test all electronics before mounting any hardware. This gives you full access to the cavity and easy debugging if something isn't working.

1. Shield cavity with copper foil tape
2. Install pickups and route wires into cavity
3. Install and wire control harness
4. Test with multimeter and bench amp
5. Install bridge
6. Install tuners and strap buttons
7. Dry-fit neck, check alignment, install ferrules
8. String up

**Best for:** First-time builders, anyone who wants to verify electronics before committing to hardware installation.

## Hardware-first

Install all mechanical hardware before any electronics. Useful if you want to verify the physical build and neck fit before investing time in wiring.

1. Install bridge
2. Install tuners and strap buttons
3. Dry-fit neck, check alignment, install ferrules
4. Verify string line and action range
5. Remove neck
6. Shield cavity, install electronics
7. Reinstall neck
8. String up

**Best for:** Builders with electronics experience who want to catch neck-fit problems early, before investing time in wiring.

## Common sequence mistakes

- **Installing ferrules before checking neck alignment** — ferrules are permanent. Check alignment first.
- **Stringing up before verifying electronics** — if there's a wiring problem you'll have to loosen strings to access the cavity.
- **Mounting the neck before installing pickups** — the neck heel can block access to the pickup cavities on some configurations.
