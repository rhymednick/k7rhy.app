Hardware installation is the process of turning a complete, cured body into a mechanically assembled guitar. Electronics are handled separately in [Wiring & Electronics](/relay/electronics/overview) — you can do hardware before or after wiring, but the order matters. See the sequence notes below.

The body should be fully cured and bonding-checkpoint-verified before this phase begins — see [Assembling the Body](/relay/build/body).

## What you're doing in this phase

- Mounting the bridge
- Installing tuning machines
- Installing strap buttons
- Dry-fitting and then permanently attaching the neck

One step is hard to undo once completed: installing the neck ferrules. Everything else is adjustable. Do your dry-fits before committing.

## Sequence: hardware before or after electronics?

There's no single correct order. Both approaches work — choose based on your situation.

### Electronics-first (recommended for first builds)

Wire and test all electronics before mounting hardware. This gives you full cavity access and makes debugging much easier.

1. Complete electronics installation and bench test ([Wiring & Electronics](/relay/electronics/wiring))
2. Mount bridge, tuners, strap buttons
3. Dry-fit neck — check alignment and string line
4. Install ferrules and bolt neck permanently
5. String up

**Best for:** First builds, or any build where you want to verify the electronics work before committing to hardware.

### Hardware-first

Verify the physical build and neck fit before investing time in wiring.

1. Mount bridge, tuners, strap buttons
2. Dry-fit neck — check alignment, string line, action range
3. If fit is good, remove neck
4. Complete electronics installation and bench test
5. Reinstall neck, install ferrules
6. String up

**Best for:** Builders with electronics experience who want to confirm neck fit early.

## The neck step

Neck attachment is the most likely failure point in this build. Do these checks before installing ferrules:

- Neck seats fully in pocket with no rocking
- String line centered on fretboard — stretch a string from nut to bridge and check both sides
- Neck angle correct — straight edge along fretboard toward bridge
- All four bolt holes align

<DecisionNote variant="caution" title="Do not install ferrules until alignment checks pass">
    Ferrules are permanent. Once pressed in, they define where the neck bolts. If something isn't right, this is the moment to address it — or to bring in a luthier for this step specifically. See [When to Call a Luthier](/relay/setup/professional).
</DecisionNote>
