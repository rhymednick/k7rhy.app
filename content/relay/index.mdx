Relay is an electric guitar you print at home on a conventional FDM 3D printer. Every model shares the same double-cut body — three pickup cavities, the same scale length, the same hardware mounting points. What makes each model different is what you put inside it: pickup selection, wiring, and selector strategy.

Building a Relay guitar is a real project. You will spend time on the printer, time bonding and finishing the body, time sourcing parts, and time with a soldering iron. The result is a playable, giggable instrument that you understand completely — because you built it. Plan on a few weekends and $200–$400 in parts depending on the model you choose.

## The platform

### One body, seven voices

All models use the same printed body. Pickup ring adapters bridge the gap between the body's humbucker-sized cavities and smaller pickup formats like lipstick or P90. You choose one body design to print, then wire it for whichever model you're building.

### 24.75" scale length

Scale length is the distance from the nut to the bridge saddles — it determines how long the vibrating portion of each string is. Relay uses 24.75 inches, the same as a standard Les Paul. Shorter than a Strat's 25.5 inches: slightly lower string tension, slightly warmer tone, and easier bends. It also means neck availability is excellent — 24.75" necks are common, well-priced, and come in a wide range of profiles and fretboard materials.

### Three pickups

Every model has three pickup cavities. What changes is how those pickups interact. A Relay model can use the middle pickup as a primary voice, an augment layer, a shaper, or part of a grouped subsystem. The selector and controls follow that role.

You don't have to install a middle pickup. The model files include optional cavity covers if you prefer a two-pickup layout.

## Switch and control hardware

The line stays coherent without forcing identical controls on every guitar. The practical rule:

- Use a **3-way selector** when the middle pickup is a **shaper** or **augment layer** added after the bridge/neck selection.
- Use a **5-way blade** when the middle pickup is a **primary voice** or part of a switched **subsystem**.

Unless bench testing says otherwise, the default passive pots for Relay models that use the **master volume + master tone** pattern are **500k audio taper, full-size** for both. Every model uses at least one humbucker or humbucker-adjacent pickup, so 500k preserves articulation and openness while still allowing you to roll off darkness.

**Push-push** fits functions you might hit while playing or that define quick-access personality. **Push-pull** fits mode-style changes where accidental activation would be annoying, or where fastest access matters less.

**Selector families today:** **3-way toggle** - Relay Lipstick, Relay Current. **Standard 5-way blade (2-pole)** - Relay Velvet, Relay Arc, Relay Torch, **Relay Reef**. **5-way Super Switch (4-pole)** - Relay Hammer concept (deeper body than a standard blade, so leave depth in the control cavity when you plan the route).

Model pages spell out each instrument’s switch positions and what the push controls do.
