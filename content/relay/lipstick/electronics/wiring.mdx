This page walks through the complete wiring process. Read [Electronics Overview](/docs/relay/lipstick/electronics/overview) first so you understand what you're building before you start soldering.

If you've never soldered before, this is a learnable skill but takes practice. I'd recommend soldering a few practice joints on scrap wire before working on the actual components.

## Tools and supplies

- Soldering iron (25–40W, fine tip)
- Rosin-core solder (60/40 or 63/37)
- Flux pen (recommended)
- Helping hands or PCB vise
- Wire strippers
- Heat shrink tubing (assorted)
- Multimeter (essential for testing)

## Testing before installation

Before installing anything in the body, test the circuit on the bench:

1. Use a multimeter to verify continuity from each pickup through to the output jack in each selector position
2. Check that no selector positions are shorted to ground
3. Plug in to an amp and tap each pickup with a screwdriver to verify signal
