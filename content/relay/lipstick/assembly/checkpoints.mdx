---
title: 'Checkpoints'
description: 'What to verify at each stage before proceeding. These catch problems while they are still fixable.'
---

These are the checks that prevent small problems from becoming expensive ones. Do each one before moving to the next stage.

## After body bonding

- [ ] All joints fully cured (minimum 24 hours, full cure per G/Flex spec)
- [ ] No visible gaps at any joint line
- [ ] Neck pocket flat and square — check with a straight edge
- [ ] Bridge mount area flat — no high spots that would prevent flush seating

## After electronics installation (before neck)

- [ ] Continuity from each pickup to output jack in each selector position
- [ ] No selector position shorted to ground
- [ ] All ground connections confirm with multimeter
- [ ] Bench test: signal from each pickup, tone and volume controls respond correctly, coil splits function on both pots
- [ ] No wires pinched between components or at cavity edges

## Neck dry-fit

- [ ] Neck seats fully in pocket with no rocking
- [ ] String line centered on fretboard — stretch a string from nut to bridge ferrule and check both sides
- [ ] Neck angle correct — check with a straight edge along the fretboard toward the bridge
- [ ] All four bolt holes align

<DecisionNote variant="caution" title="Do not install ferrules until the dry-fit passes">
Ferrules are permanent. Once installed they define where the neck bolts. Confirm everything above before you press them in.
</DecisionNote>

## After stringing

- [ ] All strings seat correctly in nut slots
- [ ] No string buzz at open position (indicates nut slot too low or nut not fully seated)
- [ ] Tuners lock and hold pitch
- [ ] All electronics function with strings installed — ground buzz reduces when touching metal parts (confirms ground path through strings and bridge)

## Before final setup

- [ ] Neck relief in acceptable range (slight bow, not backbow)
- [ ] Action measurable at 12th fret — if it's so high you can't fret notes, address before setup

If any checkpoint fails, stop and address it. The <DiscordLink /> is a good place to post photos and get guidance before proceeding.
